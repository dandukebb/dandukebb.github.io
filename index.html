Color Formats

Format Syntax Example Use Case

RGB rgb(r, g, b) rgb(67, 97, 238) Programmatic color generation
RGBA rgba(r, g, b, a) rgba(67, 97, 238, 0.5) Transparency support
HSL hsl(h, s%, l%) hsl(230, 83%, 60%) Human-readable adjustments
HSLA hsla(h, s%, l%, a) hsla(230, 83%, 60%, 0.5) Transparency + intuitive tuning
